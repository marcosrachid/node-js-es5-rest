# node-js-es5-rest

node js study

## Installation
```bash
npm init
npm install express --save
npm install body-parser --save
npm install mongodb --save
npm install connect-multiparty --save
npm install -g nodemon
```

## Execution
```bash
node app.js
```

or

```bash
nodemon app
```
